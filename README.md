# ai-ml-intern

AI & ML Intern Task: LLM Script
This project is a simple Python script that connects to the Gemini 2.5 Flash model. It takes a question from the user and returns an answer from the AI.

🚀 My Approach
I followed a simple and structured process to build this:

Modern Tools: I used the latest google-genai library because it is faster and more reliable than older versions.

Clean Code: I divided the code into two parts: one for talking to the AI and one for interacting with the user. This makes it easy to read.

Security: I used a .env file to hide my API Key. This is a professional practice to ensure private keys are not leaked on GitHub.

Error Handling: I added a "try-except" block so that if the internet fails or the API key is wrong, the program shows a helpful message instead of crashing.

🛠️ Tech Stack
Language: Python

AI Model: Gemini 2.5 Flash

Library: google-genai
